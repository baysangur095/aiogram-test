Telegram-бот для заметок на aiogram 3. Проект написан в рамках 
глубокого изучения фреймворка: FSM, инлайн-клавиатуры, middleware, 
асинхронная работа с БД.

## Функционал
- Добавление заметок через FSM-диалог
- Список и удаление заметок инлайн-кнопками (callback + edit_text)
- Хранение в SQLite (aiosqlite), состояния FSM — в Redis
- Антиспам через кастомный Throttling Middleware
- Валидация пользовательского ввода
